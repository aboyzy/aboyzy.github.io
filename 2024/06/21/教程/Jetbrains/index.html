Jetbrains系列激活

稚
2024-06-21
教程

做个激活攻略方便分享

下载软件

这个就不用说了吧直接去官网 https://www.jetbrains.com/zh-cn/ 点击 开发者工具要下哪个直接下。

下载老版本

  • IDEA 2023.2.2
  • PyCharm 2023.2.2
  • DateGrip 2023.2.2

破解

打开网站jet.fsyl.eu.org点击最上方的蓝色链接下载解压包

下载完成之后解压文件进入 scripts文件夹

[warning]注意：这里一定要把文件解压到一个文件夹，不要直接在压缩包里打开，而且这个文件夹不要删除，删除了激活就无效了[/warning]

点击 install-all-users.vbs文件

直到出现弹窗down出现再打开idea

进入上方的网站 点击对应的软件即可复制对应软件的解压密钥

在激活方式中选择 Activation code 填写激活密钥
