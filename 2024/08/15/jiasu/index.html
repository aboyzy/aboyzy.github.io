稚

一些之前写的和加速镜像链接 欢迎提供更多加速地址

加速链接

镜像站点

  • 阿里巴巴开源镜像站：https://mirrors.aliyun.com
  • 腾讯软件源：https://mirrors.cloud.tencent.com
  • 华为开源镜像站：https://mirrors.huaweicloud.com

前端公共CND

国外经典CDN

  • jsdelivr：https://www.jsdelivr.com
  • unpkg：https://unpkg.com
  • cdnjs：https://cdnjs.com
  • cnd.js国内代理:https://cdn.zstatic.net/

国内用推荐

  • 饿了么：https://github.elemecdn.com、https://npm.elemecdn.com
  • 静态资源：https://www.zstatic.net/

可用但不全

  • 字节跳动静态资源公共库：https://cdn.bytedance.com
  • Staticfile CDN：https://staticfile.org

NPM源

  • 阿里云：https://registry.npmmirror.com

Python 镜像源

  • 阿里云：https://mirrors.aliyun.com/pypi/simple

  • 清华大学：https://pypi.mirrors.ustc.edu.cn/simple

Maven 镜像仓库

  • 阿里云：https://maven.aliyun.com/mvn/guide
  • 腾讯云：https://mirrors.cloud.tencent.com/help/maven.html

Github 下载加速

  • https://mirror.ghproxy.com
  • 游猴插件:https://greasyfork.org/zh-CN/scripts/412245

vercel 加速

  • 如果你的网站部署在vercel

    ，则把 cname 记录改为：

    • vercel-cname.xingpingcn.top

  • 如果你的网站部署在netlify

    ，则把 cname 记录改为：

    • netlify-cname.xingpingcn.top

  • 如果你的网站部署在netlify和vercel

    上，则把 cname 记录改为：

    • verlify-cname.xingpingcn.top
