浪潮元脑服务器NF5280M6安装Windows server 2016

BMC管理地址配置（选配）

开机按del键进入bios

选择Server mgmt

选择BMC network configuration 里边的最后一个选项 配置静态选择 Manual和static 配置地址掩码和网关 （网关一定要写不然不成功）

然后按esc返回 会出现Set BMc Network Config 选择是 然后回再次弹窗提示是否配置成功 然后f10 保存重启就可以

配置Read1

客户要求两块盘做raid1 使用 在高级设置Advanced中找到阵列卡 一般为最后

选择Main menu

选择configuration Management

因为我们是新设备没有做配置所以选择create virtual drive 创建虚拟卷

选择raid类型

选择Select Drive 回车进入选择硬盘界面

选择你要的组raid的硬盘设置为enabled

然后到最下边选择apply change

会进行两次确认然后回到raid配置界面 配置写策略（这个为啥也不知道 官方文档推荐做）

然后保存会进行两次确认

安装系统

将系统盘插上后 重启系统 按F11 进入选择u盘启动器进行安装 应该提前将raid卡驱动启动拷贝到u盘上 因为安装时可能无法识别raid的硬盘需要手动导入 Raid卡驱动 选择外部插件 ->Raid卡 选择自己的版本 下载解压到u盘 在安装系统参考这个文档 安装完成之后还需要安装对应的网卡驱动和芯片组驱动 搜索对应版本驱动下载
