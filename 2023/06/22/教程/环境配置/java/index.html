Tomcat 安装

下载

下载Tomcat9 因为我用的jdk1.8所以下载9 下载之前请先看你的jdk是否支持该版本tomcat

镜像下载 (推荐)
打开镜像链接https://mirrors.tuna.tsinghua.edu.cn/apache/tomcat/ 下载自己对应版本的tomcat
我使用9.0.83 进入链接之后点击bin文件夹根据系统进行下载 这里我采用的windows 所以下载apache-tomcat-9.0.83-windows-x64.zip

官网下载
打开官网:https://tomcat.apache.org/download-90.cgi

安装

Tomcat 运行依赖java所以请提前配置好java环境

安装好之后将压缩包里的内容解压出来进入解压出来的文件夹的bin目录，找到目录b下的startup.bat点击就可以启动tomcat

中文乱码

打开tomcat安装目录/conf/logging.properties文件，搜索java.util.logging.ConsoleHandler.encoding = UTF-8(第51行) 将UTF-8更改为GBK 然后重新启动就可以了
